## 專案目標

建立一個 CNN 權重資料庫，讓不同任務的模型可以互相分享學習到的權重，實現**遷移學習**。

### 核心功能
- 儲存訓練好的 CNN 權重到向量資料庫
- 根據圖片特徵或模型架構進行相似度搜尋
- 新任務可以從資料庫中取得相似類別的權重作為初始化

## 資料庫架構

### Qdrant Collection Schema

使用**單一 collection** 儲存所有類別的權重，藉由 `category` 欄位區分。

#### Collection: `cnn_weights`

```python
{
    "vectors_config": {
        "size": 512,           # 特徵向量維度
        "distance": "COSINE"   #餘弦相似度
    }
}
```

#### Payload 結構

| 欄位 | 類型 | 說明 |
|------|------|------|
| `model_name` | str | 模型名稱（如 "SimpleCNN"） |
| `category` | str | 物體類別（如 "caterpillar"） |
| `epoch` | int | 訓練回合數 |
| `accuracy` | float | 準確率 |
| `weights` | dict | 完整權重（JSON 格式） |
| `image_stats` | dict | 圖片統計特徵（mean, std） |

#### 儲存方式

1. **Vector（用於 HNSW 搜索）**：512 維特徵向量
   - 模型特徵：每層的 mean, std, min, max（4 features × 層數）
   - 圖片特徵：每張圖的 mean, std, min, max + histogram（12 features × 50 張）

2. **Payload（完整權重）**：無大小限制
   - 所有層的權重以 JSON 格式存儲
   - 可完整還原模型權重

### Schema 圖示

```
Collection: cnn_weights
├── Vector (512 dim, COSINE distance)
│   └── 特徵向量 [model_features + image_features]
└── Payload
    ├── model_name: "SimpleCNN"
    ├── category: "caterpillar"
    ├── epoch: 3
    ├── accuracy: 59.5
    ├── weights: {"conv1.weight": [...], "fc1.bias": [...], ...}
    └── image_stats: {"mean": 0.45, "std": 0.22, "num_samples": 50}
```

### 使用流程

#### 第一輪訓練（冷啟動 + 儲存）

```python
# 1. 建立 Vault
vault = QdrantWeightVault(url=..., api_key=..., collection_name="cnn_weights")

# 2. 創建模型（使用 He init）
model = create_db_cnn(vault, num_classes=1)

# 3. 訓練
train_model(model)

# 4. 存到 vault
model_state = {name: param.data for name, param in model.named_parameters()}
image_sample = [train_dataset[i][0] for i in range(50)]

vault.store_category_weights(
    model_state=model_state,
    model_name="SimpleCNN",
    category="caterpillar",
    epoch=3,
    accuracy=59.5,
    dataset_sample=image_sample
)
```

#### 第二輪訓練（Vault Init）

```python
# 1. 用圖片搜索 vault
image_sample = [new_dataset[i][0] for i in range(50)]
best_category, similarity = vault.find_similar_category_by_images(image_sample)

# 2. 直接從 vault 創建模型（不做 He init）
if best_category and similarity > 0.3:
    vault_weights = vault.get_category_weights(best_category)
    model = create_db_cnn_from_vault(vault_weights, num_classes=1)
else:
    # 沒有相似類別，使用冷啟動
    model = create_db_cnn(vault, num_classes=1)

# 3. 繼續訓練
train_model(model)
```

### 特徵提取

#### 模型特徵（512 維）

每層提取 4 個統計值（mean, std, min, max），不足則補 0。

```python
def _extract_features(model_state):
    features = []
    for name in sorted(model_state.keys()):
        tensor = model_state[name]
        arr = tensor.flatten()
        features.extend([arr.mean(), arr.std(), arr.min(), arr.max()])
    return pad_or_truncate(features, 512)
```

#### 圖片特徵（512 維）

每張圖片提取 12 個特徵（mean, std, min, max + 8-bin histogram），最多 50 張圖。

```python
def _extract_image_features(images):
    features = []
    for img in images[:50]:
        img_flat = img.flatten()
        features.extend([
            img_flat.mean(), img_flat.std(),
            img_flat.min(), img_flat.max()
        ])
        # 8-bin histogram
        hist = torch.histc(img_flat, bins=8)
        features.extend(hist / img_flat.numel())
    return pad_or_truncate(features, 512)
```

## 新功能：圖片特徵搜索

### 概念

不同於傳統的模型架構匹配，我們直接分析訓練資料集的圖片特徵，來找到相似的已訓練類別。

### 實現方式

1. **儲存時**：從訓練資料集抽樣 50 張圖片，提取統計特徵，存到 vault
2. **搜索時**：從新資料集抽樣 50 張圖片，提取相同特徵，用 HNSW 搜索最相似類別

### 優點

- 不依賴模型架構
- 直接根據資料特性匹配
- 適用於遷移學習場景

### 限制

- 統計特徵較簡單，可能無法捕捉深層語意
- 未來可考慮使用預訓練模型（如 ResNet）提取更深層特徵
