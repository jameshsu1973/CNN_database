## 專案目標

建立一個 CNN 權重資料庫，讓不同任務的模型可以互相分享學習到的權重，實現**遷移學習**。

---

## 今日重點

**目標**：實現通用的權重存取方式，讓任意 PyTorch 模型都能使用向量資料庫。

**問題**：原本的 `DBConv2d` / `DBLinear` 類需要繼承修改模型定義，不夠泛化。

**解決方案**：
- 訓練後：從任意模型提取 `state_dict`，存儲到 vault
- 初始化時：從 vault 檢索權重，載入到相同架構的模型

---

## 泛化權重管理系統

### 核心概念

```
任意 PyTorch 模型 → extract_weights() → Qdrant Cloud
                                    ↓
任意 PyTorch 模型 ← load_weights() ← Qdrant Cloud
```

### 關鍵特性

1. **層類型無關**：不管裡面是 Conv2d、Linear、BatchNorm、Embedding 都能處理
2. **模型無關**：支持 ResNet、VGG、MobileNet、自定義模型
3. **無需修改模型**：直接操作 `state_dict`，不需繼承特定類別
4. **自動指紋壓縮**：大維度層（>65535）自動使用指紋壓縮

## 實作改動

### 1. 新增 `cnn_weight_vault/wrap.py`

提供通用權重管理函數：
- `extract_weights(model, vault, model_name, epoch, accuracy, object_category)` - 存儲權重
- `load_weights(model, vault, object_category, force)` - 載入權重
- `save_model_to_vault()` / `load_model_from_vault()` - 整體模型操作
- `TrainingHook` - 訓練鉤子，自動存儲

### 2. 修改 `qdrant_vault.py`

新增方法：
- `_store_raw(vector, layer_key, metadata)` - 通用存儲（支持指紋壓縮）
- `_get_weights_by_key(layer_key, shape, object_category, force)` - 通用檢索

修復問題：
- 指紋壓縮：大維度層自動壓縮到 32768 維
- API 兼容性：修復 `config.vectors` vs `vectors_config` 問題
- 向量轉換：確保 list 轉為 numpy array

### 3. 更新 `__init__.py`

導出新模組：

## 使用方法

### 基本用法

```python
from cnn_weight_vault import QdrantWeightVault, extract_weights, load_weights

# 1. 創建 vault
vault = QdrantWeightVault(collection_name="my_vault")

# 2. 訓練後存儲權重
extract_weights(
    model=trained_model,      # 任意 PyTorch 模型
    vault=vault,
    model_name="resnet18",
    epoch=10,
    accuracy=95.5,
    object_category="cat"
)

# 3. 初始化時載入權重
new_model = models.resnet18(weights=None)
results = load_weights(
    model=new_model,
    vault=vault,
    object_category="cat",
    force=True  # True: 強制載入, False: 相似度閾值內才載入
)
# results = {'conv1.weight': True, 'layer1.0.conv1.weight': True, ...}
```

### 支持的模型

- torchvision: ResNet, VGG, MobileNet, EfficientNet
- 自定義模型：任意 PyTorch 模型

### 訓練鉤子

```python
from cnn_weight_vault import TrainingHook

hook = TrainingHook(vault, model_name="resnet18", save_interval=5)
hook.register(model)

for epoch in range(num_epochs):
    train(model)
    hook.on_epoch_end(model, epoch, accuracy)
```

## 技術細節

### 指紋壓縮 (Fingerprint)

當權重向量維度超過 Qdrant 免費版限制（65535）時：
- 使用 Count Sketch 演算法壓縮到 32768 維
- 存儲時標記 `use_fingerprint=True`
- 檢索時重建近似值

### 層鍵值生成

根據形狀生成唯一鍵值：
- `(64, 3, 7, 7)` → `conv_64_3_7_7`
- `(512, 512)` → `linear_512_512`
- `(64,)` → `bias_64` 或 `1d_64`

### 相似度檢索

- **force=True**: 直接獲取最新存儲的權重
- **force=False**: 使用餘弦相似度，僅在相似度 > 閾值時載入
